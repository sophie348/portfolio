Cloud Computing Services Deploiement

Cloud Computing

Le Cloud Computing est devenu incontournable , c'est une véritable révolution dans le monde informatique.Le Cloud Computing fournit des services ou des applications informatiques en ligne , accessible à tout instant depuis n'importe quel appareil(smartphone, PC de bureau, ordinateur portable).Le Cloud Computing permet de partager une infrastructure, une application (comptable par exemple) ou une plateforme.Il ne faut plus avoir le programme dans l'ordinateur , seulement une connexion à internet et le fournisseur s'occupe de tout.


Le Cloud est la fourniture de services informatiques , ce domaine peut toucher les serveurs , les bases de données, les logiciels donc tout ce qui peut se gérer à distance avec l'internet.



Quels sont les avantages?

  • Accessibilité
  • Reduction des couts
  • Securité
  • Disponible 24h/24

Quels sont les inconvenients?

  • Dependance à la connexion internet
  • Dépendance au prestataire

Les differents services

Iaas

Infrastructure as a service

C'est une structure qui consiste à proposer un parc informatique virtualisé,un espace de stockage,une technologie réseau.Le consommateur peut installer des applications et un système d'exploitation sur les machines virtuelles. Il est ainsi dispensé d'acheter du matériel informatique, ce service fonctionne comme un centre de traitement de données(datacenter)

Paas

Plateform as a service

Le systeme d'exploitation et les outils d'infrastructure sont sous la reponsabilité du fournisseur. Le consommateur a le controle des applications et peut rajouter ses propres outils.C'est l'idéal pour les developpeurs

Saas

Software as a service

Les applications sont mises à disposition.Elles peuvent être manipulées à l'aide d'un navigateur web ou installées sur le PC.

Le deploiement

Il y a quatre formes de deploiement.

  1. Cloud privé

    C'est la forme la plus répandue.Les données peuvent être hébergées par l'entreprise ou par un tiers.

  2. Cloud communautaire

    Il est utilisé par différentes organisations avec des besoins communs.Par exemple plusieurs organisations font un cloud pour gérer une application.

  3. Cloud public

    Il est accessible depuis l'internet et plusieurs organisations utilisent des ressources identiques(par exemple Microsoft Azure,Google cloud)

  4. Cloud mixte

    Il utilise à la fois le cloud privé et public
