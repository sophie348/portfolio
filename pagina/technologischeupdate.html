Cloud Computing

Cloud Computing is essentieel geworden, het is een ware revolutie in de IT-wereld. Cloud Computing biedt online IT-diensten of applicaties die op elk moment toegankelijk zijn vanaf elk apparaat (smartphone, desktop-pc, laptop...). Cloud Computing maakt het mogelijk om een ​​infrastructuur, een applicatie (bijvoorbeeld boekhouding) of een platform. Je hoeft het programma niet meer in de computer te hebben, alleen een internetverbinding en de leverancier regelt alles.


De cloud is het leveren van computerdiensten, dit gebied kan van invloed zijn op servers, databases, software, dus alles dat op afstand kan worden beheerd met internet.



Wat zijn de voordelen?

  • Toegankelijkheid
  • Kostenbesparing
  • Veiligheid
  • 24 uur per dag bereikbaar

Wat zijn de nadelen?

  • Afhankelijkheid van internetverbinding
  • Afhankelijkheid van provider

De verschillende diensten

Iaas

Infrastructure as a Service

Het is een structuur die bestaat uit het aanbieden van een gevirtualiseerd computerpark, opslagruimte, netwerktechnologie en de klant kan applicaties en een besturingssysteem op de virtuele machines installeren. Het is dus vrijgesteld van de aankoop van computerhardware, deze dienst functioneert als een dataverwerkingscentrum (datacenter)

Paas

Plateform as a Service

Het besturingssysteem en de infrastructuurtools vallen onder de verantwoordelijkheid van de leverancier. De consument heeft de controle over de toepassingen en kan zijn eigen hulpmiddelen toevoegen. Het is ideaal voor ontwikkelaars

Saas

Software as a Service

De toepassingen zijn beschikbaar. Ze kunnen worden gemanipuleerd met behulp van een webbrowser of worden geïnstalleerd op de pc.

Implementatie

Er zijn vier vormen van implementatie

  1. Private Cloud

    Dit is de meest voorkomende vorm; gegevens kunnen worden gehost door het bedrijf of door een derde partij.

  2. Community Cloud

    Het wordt gebruikt door verschillende organisaties met gemeenschappelijke behoeften, verschillende organisaties maken bijvoorbeeld een cloud om een ​​applicatie te beheren.

  3. Openbare Cloud

    Het is toegankelijk vanaf internet en verschillende organisaties gebruiken identieke bronnen (bijv. Microsoft Azure, Google cloud)

  4. Hybrid Cloud

    Het maakt gebruik van zowel de privé- als de openbare cloud
